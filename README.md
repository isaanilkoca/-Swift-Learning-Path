# Swift & iOS Intermediate Practices

Welcome to my Swift learning path and intermediate practices repository! This repository serves as a centralized archive for various applications, mini-games, and technical implementation experiments designed to master the core capabilities of iOS development, Object-Oriented Programming (OOP), and native Apple frameworks.

## 🕹️ Featured Projects & Mini-Games

Within this learning track, I have explored various interactive logic layouts and graphics capabilities:

*   **Hunt for Gollum & Interactive Games:** Implemented text-based or interactive gameplay mechanics focusing on state machine logic, conditional workflows, and user choice persistence in Swift.
*   **ARKit & Spatial Computing (SolarSystemByArKit / ArKitFirstProject):** Explored augmented reality pipelines by initializing tracking sessions, placing 3D objects, and rendering spatial celestial animations in real-world coordinates using **ARKit** and **SceneKit**.
*   **ArtShots & Media UI:** Practiced layout designs, custom asset integration, and image display workflows using native UIKit hierarchies.
*   **System Customization (HodorKeyboard / DarkModeApp):** Experimented with app extensions by implementing custom software keyboard layouts and managing app-wide visual traits via dynamic dark mode trait configurations.

## 🧠 Core Technical Concepts Covered

This repository acts as a comprehensive laboratory for sharpening foundational iOS development skills:
*   **Object-Oriented Programming (OOP):** Deep-dive into class structures, inheritance, encapsulation, polymorphism, and protocol-oriented programming.
*   **UIKit Layout Mechanics:** Hands-on experience with Storyboards, view hierarchies, life-cycle methods (`viewDidLoad`, `viewWillAppear`), and delegation patterns.
*   **State & Asset Management:** Structuring application logic cleanly to handle asset catalogs, application states, and localized memory management safely.
